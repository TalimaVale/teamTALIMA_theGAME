1. Import Standard Assets https://www.assetstore.unity3d.com/en/#!/content/32351
2. Drag HolyShift/Prefabs/Water.prefab to your scene
3. Tweak Waves density, Distortion power, Max threshold, Threshold density, Threshold falloff, Timescale, Waves colour and Waves target value parameters in HolyShift/Materials/LowPolyWater.material
4. Enjoy cool water effects!
